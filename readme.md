## How to run the project

1. Clone the repository
2. Run `yarn install`
3. Add a `.env` file in the root of the project with the following variables:
    - `EXPO_PUBLIC_API_URL`
    - `EXPO_PUBLIC_API_KEY`
4. Run `yarn start`
5. Run `yarn ios` or `yarn android`
